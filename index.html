Type me well.

a CSS typography challenge

X-Ray Goggles

letter

Challenge 1:

Adjust the space between the letters in the word above to make it appear more visually pleasing and balanced. Typographers call this kerning. In web design, you can do this by adjusting pixels under the CSS property letter-spacing. Get started by clicking the pink X-Ray Goggles button. Hover over the part of the type that needs to be changed and then change the CSS by clicking C and the spacebar at the same time. When you finish the challenge click the submit button.
